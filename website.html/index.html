Profile

吉沢 亮

日本の俳優。東京都出身。アミューズ所属。
生年月日 : 1994年2月1日生まれ(29歳)。
血液型 : B型。
身長 : 171cm。

人物

趣味はギターの練習。
特技は剣道で小学校から高校までの約9年間続けていた。剣道は二段の腕前を持っている。
また、キックボクシングの経験もある。

出演

2009年に行われた『アミューズ全国オーディション2009 THE PUSH!マン』で審査員特別賞を受賞。
2010年、舞台『BLACK PEARL』にて、ラム役で舞台初出演し、俳優デビューした。
『仮面ライダーフォーゼ』で注目を集める。以降、ドラマ、映画、舞台で活躍。
